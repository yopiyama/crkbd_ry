# Corne Cherry キーマップ

Corne Cherry の自分用キーマップです。

## Base

```txt
,-----------------------------------------.                  .-----------------------------------------,
|  Tab |   Q  |   W  |   E  |   R  |   T  |                  |   Y  |   U  |   I  |   O  |   P  | - _  |
|------+------+------+------+------+------|                  |------+------+------+------+------+------|
| Ctrl |   A  |   S  |   D  |   F  |   G  |                  |   H  |   J  |   K  |   L  | ; :  | ' "  |
|------+------+------+------+------+------|                  |------+------+------+------+------+------|
| Shft |   Z  |   X  |   C  |   V  |   B  |                  |   N  |   M  | , <  | . >  | / ?  | = +  |
`---------------------------+------+------+------.    .------+------+----------------------------------'
                            |  ALT | GUI_T|  SYM |    |  NUM | BSCP | Space|
                            `--------------------'    `--------------------'
```

| Key   | Tap   | Hold      |
| ----- | ----- | --------- |
| NUM   | Enter | NUM Layer |
| SYM   | Space | SYM Layer |
| GUI_T | ESC   | Left GUI  |

## NUM

```txt
,-----------------------------------------.                  .-----------------------------------------,
|      |   1  |   2  |   3  |   4  |   5  |                  |   +  |   -  |   *  |   /  |   =  |  ENT |
|------+------+------+------+------+------|                  |------+------+------+------+------+------|
|      |   6  |   7  |   8  |   9  |   O  |                  |   ←  |   ↓  |   ↑  |   →  |      |      |
|------+------+------+------+------+------|                  |------+------+------+------+------+------|
|      |      |      |      |      |      |                  |  Del |      |   ,  |   .  |      |      |
`---------------------------+------+------+------.    .------+------+----------------------------------'
                            |      |      | LEDS |    |      |      |      |
                            `--------------------'    `--------------------'
```

## SYM

```txt
,-----------------------------------------.                  .-----------------------------------------,
|      |  F1  |  F2  |  F3  |  F4  |  F5  |                  |   !  |   @  |   #  |   $  |   %  |      |
|------+------+------+------+------+------|                  |------+------+------+------+------+------|
|      |  F6  |  F7  |  F8  |  F9  |  F10 |                  |   ^  |   &  |   `  |   ~  |   \  |   |  |
|------+------+------+------+------+------|                  |------+------+------+------+------+------|
|      |  F11 |  F12 |      |      |      |                  |   (  |   )  |   [  |   ]  |   {  |   }  |
`---------------------------+------+------+------.    .------+------+----------------------------------'
                            |      |      |      |    | LEDS | Conv |      |
                            `--------------------'    `--------------------'
```

Conv = Ctrl + Space
